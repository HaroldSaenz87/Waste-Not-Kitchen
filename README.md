# Waste Not Kitchen (WNK)

---

## Overview
**Waste Not Kitchen** is a web application that helps reduce food waste by connecting restaurants with surplus meals to customers, donors, and needy individuals.

The platform allows users to:
- Browse available plates
- Reserve & donate orders
- Access admin reporting tools

---

## User Roles
The system supports **five types of users**:

- **Administrators** - View member info & generate reports
- **Restaurants** - Post and manage surplus plates
- **Customers** - Reserve and purchase plates
- **Donors** - Pay for plates to donate
- **Needy Users** - Reserve donated plates
